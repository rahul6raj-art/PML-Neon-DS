## Content shape

Each item in `items` should include:

- **`title`** — headline (14px medium)
- **`badges`** — `{ label, context }` where `context` is `positive` | `negative` | `notice` | `default`
- **`body`** — summary paragraph
- **`time`** — e.g. `25 Jan, 10:12 AM`
