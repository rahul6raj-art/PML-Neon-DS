## Behaviour

- **SVG line** — Multi-layer blurred glow plus a 1.5px stroke; gradients use design tokens and respect light/dark theme. Optional toggles turn off glow, ripple, or dot for minimal variants.
- **svgPath** — Demos use legacy PortfolioHome cubics (M and C commands). The areaFillPath prop is optional for data symmetry. Gain charts default to no under-line gradient (blur and stroke only). Loss charts default to the soft vertical fill; override with showAreaFill. The path must end at the ripple so the line meets the pulse/dot (last point uses indicatorXRatio times viewBoxWidth for X and indicatorY for Y).
- **Endpoint** — Rendered in HTML so it does not stretch when the SVG uses preserveAspectRatio with value none.

## Playground

Use **Positive return** to switch between gain (teal / green gradients) and loss (orange / red) styling. Demo paths and indicator position update automatically.

**Appearance** controls:

- **Gradient + glow** — Full gradient stroke + multi-layer blur; off = a single solid line using the accent stop token (`--gw-line-3`).
- **Ripple (pulse)** — Outer animated pulse ring.
- **Dot (inner ring + center)** — Inner ring and center dot.
